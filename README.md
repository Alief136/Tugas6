**Instalasi Proyek**

**1. Clone Proyek dari GitHub**
```bash
git clone https://github.com/Alief136/Tugas6.git

cd Tugas6
```  
**2. Install Dependensi Proyek**  
```bash
composer install
```  

**3. Jalankan Server**  
Jalankan server lokal menggunakan perintah:  
```bash
php spark serve
```  

  

**4. Akses di Browser**  
Buka browser dan akses URL berikut:  
```plaintext
http://localhost:8080/
```
